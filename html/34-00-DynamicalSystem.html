dynamical system


A dynamical system on X where X is an open subset of ℝn is a differentiable map

ϕ:ℝ×X→X

where

ϕ⁢(t,𝐱)=ϕt⁢(𝐱)

satisfies

  • i

    ϕ0⁢(𝐱)=𝐱 for all 𝐱∈X (the identity function)

  • ii

    ϕt∘ϕs⁢(𝐱)=ϕt+s⁢(𝐱) for all s,t∈ℝ (composition)

[HSD][PL]

Note that a planar dynamical system is the same definition as above but with X an open subset of ℝ2.
