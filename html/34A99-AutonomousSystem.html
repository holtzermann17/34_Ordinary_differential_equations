autonomous system


A system of ordinary differential equation is autonomous when it does not depend on time (does not depend on the independent variable) i.e. x˙=f⁢(x). In contrast nonautonomous is when the system of ordinary differential equation does depend on time (does depend on the independent variable) i.e. x˙=f⁢(x,t).

It can be noted that every nonautonomous system can be converted to an autonomous system by adding a dimension. i.e. If 𝐱˙=𝐟⁢(𝐱,t) 𝐱∈ℝn then it can be written as an autonomous system with 𝐱∈ℝn+1 and by doing a substitution with xn+1=t and x˙n+1=1.
