Cauchy initial value problem


Let D be a subset of ℝn×ℝ, (x0,t0) a point of D, and f:D→ℝ be a function.

We say that a function x⁢(t) is a solution to the Cauchy (or initial value) problem

{x′⁢(t)=f⁢(x⁢(t),t)x⁢(t0)=x0 (1)

if

  1. 1.

    x is a differentiable function x:I→ℝn defined on a interval I⊂ℝ;

  2. 2.

    one has (x⁢(t),t)∈D for all t∈I and t0∈I;

  3. 3.

    one has x⁢(t0)=x0 and x′⁢(t)=f⁢(x⁢(t),t) for all t∈I.

We say that a solution x:I→ℝn is a maximal solution if it cannot be extended to a bigger interval. More precisely given any other solution y:J→ℝn defined on an interval J⊃I and such that y⁢(t)=x⁢(t) for all t∈I, one has I=J (and hence x and y are the same function).

We say that a solution x:I→ℝn is a global solution if D⊂=ℝn×I.

We say that a solution x:I→ℝn is unique if given any other solution y:I→ℝn one has x⁢(t)=y⁢(t) for all t∈I (i.e. x is the unique solution defined on the interval I).

0.1 Notation

Usually the differential equation in (1) is simply written as x′=f⁢(x,t). Also, depending on the topics, the name chosen for the function and for the variable, can change. Other common choices are y′=f⁢(y,t) or y′=f⁢(y,x). It is also common to write x˙=f⁢(x,t) when the independent variable represents a time value.
